# 📚 台灣全學段題庫測驗

免費線上題庫測驗網站，涵蓋台灣 **國小、國中、高中、大學** 共 116 個科目、**2,320 題**。
題目由教師原創編寫（國中小高中依 **108 課綱**；大學為各系基礎課程），每題附詳解。

🔗 **線上使用**：https://sancola1219-collab.github.io/taiwan-exam-bank/

## 三種挑戰模式

| 模式 | 玩法 |
|------|------|
| 📝 一般練習 | 自選難度與題數，可前後翻題慢慢想 |
| ⚡ 搶答挑戰 | 每題限時 15 秒，逾時算錯，連續答對累積連對數 |
| 🏆 闖關模式 | 基礎→中等→挑戰三關，每關 5 題答對 4 題才能晉級 |

## 功能

- 選學段 → 科目年級 → 挑戰模式 → 作答 → 自動改分 + 逐題詳解
- 🔄 優先出你沒做過的題目，題庫更新後自動抓新題
- 📒 錯題本：答錯自動收錄，可重考，答對即移除
- 📊 成績紀錄、⏱️ 計時器、🔗 一鍵分享、🌙 深色模式
- 手機、平板、電腦皆可使用，免安裝、免登入、完全免費

## 科目涵蓋

- **國小**（1–6 年級）：國語、數學、英語、自然、社會、生活
- **國中**（會考方向）：國文、英文、數學、生物、理化、地科、歷史、地理、公民
- **高中**（學測方向）：國文、英文、數學、物理、化學、生物、地科、歷史、地理、公民
- **大學**（8 大學群 46 科）：
  - 理學群：微積分、線性代數、統計學、普通物理、普通化學、有機化學、生物學、地球科學
  - 工學群：工程數學、靜力學、熱力學、流體力學、材料科學、電路學
  - 資訊學群：計算機概論、程式設計、資料結構、資料庫系統、計算機網路、作業系統、演算法、離散數學
  - 商管學群：經濟學原理、總體經濟學、會計學、管理學、行銷學、財務管理
  - 法政學群：法學緒論、民法概要、刑法概要、憲法、政治學、公共行政、國際關係
  - 醫護學群：解剖生理學、營養學、藥理學、公共衛生
  - 人文社科學群：心理學、社會學、哲學概論、教育學概論、環境科學
  - 語文學群：大學英文、基礎日語

## 技術

純靜態網站（HTML/CSS/JS + JSON 題庫），部署於 GitHub Pages。
數學公式以 KaTeX 渲染。個人資料只存在使用者自己的瀏覽器（localStorage）。

## 開發

```bash
node dev-server.js   # 本機預覽 http://localhost:8735
node validate.js     # 驗證題庫格式並產生 data/manifest.json
```

題庫檔：`data/{學段}_{科目}_{年級}.json`，格式見 `docs/design.md`。
